📤 DÉPLOIEMENT EN 5 MINUTES
═══════════════════════════════════════════════════════════════

1. Allez sur : https://www.netlify.com/

2. Cliquez "Sign up" → Connectez avec GitHub

3. Cliquez "Add new site" → "Import from GitHub"

4. Sélectionnez votre repo : Jeho05/gamezone

5. CONFIGURATION :
   • Base directory : createxyz-project/_/apps/web
   • Build command : npm run build
   • Publish directory : createxyz-project/_/apps/web/build/client

6. Cliquez "Deploy site"

7. ATTENDEZ 2-3 MINUTES

8. Testez l'URL donnée : https://gamezone-xxxx.netlify.app

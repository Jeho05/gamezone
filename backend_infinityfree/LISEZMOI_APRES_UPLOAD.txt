🆘 DÉPANNAGE
═══════════════════════════════════════════════════════════════

Erreur 500 Internal Server Error ?
→ Vérifiez que le fichier .env existe dans /htdocs/api/
→ Vérifiez les identifiants MySQL dans .env

Erreur 404 Not Found ?
→ Vérifiez que les fichiers sont bien dans /htdocs/ (pas dans un sous-dossier)
→ Vérifiez que le dossier api/ existe

Erreur CORS ?
→ Mettez à jour le .htaccess avec l'URL Vercel exacte
→ Redémarrez le serveur (ou attendez 2 min)

Connexion base de données échoue ?
→ Vérifiez les identifiants MySQL dans .env
→ Testez la connexion dans phpMyAdmin

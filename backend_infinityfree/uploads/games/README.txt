Ce dossier contient les images des jeux uploadées par les administrateurs.

Les images sont automatiquement :
- Redimensionnées si trop grandes (max 1200px de largeur)
- Optimisées pour le web
- Nommées de façon unique (game_xxxxx.ext)

Formats supportés : JPG, PNG, GIF, WEBP
Taille maximale : 5MB par image

Les images sont accessibles via URL :
http://localhost/projet%20ismo/uploads/games/[nom_fichier]

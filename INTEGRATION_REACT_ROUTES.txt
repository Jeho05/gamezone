MODIFICATION OPTIONNELLE: Page Shop

Fichier: createxyz-project/_/apps/web/src/app/player/shop/page.jsx

Après un achat réussi, ajouter un bouton pour voir la facture:

// Dans la fonction après création d'achat réussi
if (data.success && data.purchase) {
  toast.success('Achat créé avec succès !', {
    action: {
      label: 'Voir Ma Facture',
      onClick: () => navigate('/player/my-invoices')
    }
  });
}

Ou ajouter un lien dans la modal de confirmation:

<button
  onClick={() => navigate('/player/my-invoices')}
  className="px-4 py-2 bg-purple-600 text-white rounded-lg hover:bg-purple-700"
>
  📱 Voir Ma Facture QR
</button>
